Generador de letras y melodías de canciones con GPT-2 y Audiocraft.

Python Pandas Scikit-Learn HuggingFace GPT-2 AudioCraft Flask

Desarrollo de un generador de letras de canciones basadas en un prompt inicial, un estilo, un nombre de canción y un artista (o grupo). La letra se podrá elegir al gusto del usuario, y cuando encuentre una apropiada, la aplicación generará una melodía acorde a esa letra. Se aplicaron técnicas de fine-tuning en modelos GPT-2 para que finalmente estuvieran a disposición 3 modelos según el género: Rock, Pop y Rap. Todos entrenados en inglés. La libreria AudioCraft (MusicGen) está preentrenada para crear melodías basándose en un prompt, que en este caso, será la letra generada. Todo este proceso se ha integrado en una aplicación con Flask para facilitar la interacción del usuario.
